# Bem-vindo à Avaliação de Qualidade

Este é o repositório principal contendo a documentação e os artefatos do projeto de avaliação de qualidade do software **Mural UnB**.

Nosso objetivo é garantir que a plataforma seja tecnicamente robusta e segura para os estudantes, conduzindo avaliações metódicas baseadas nas normas da família SQuaRE (ISO/IEC 25010).
